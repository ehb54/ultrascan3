SOMO MALS Module Options Parameters Settings

Last updated: August 2024

In this panel, you can set/modify several parameters that are used in the MALS module to convert "raw" R(θ) data to an absolute scale.

  • Lambda [nm]: is the wavelength of the MALS photometer light source. Default: 664 nm.

  • Solvent refractive index: enter here the measured or calculated (for instance using SEDNTERP3) refractive index of your solvent at the temperature used for the measurements and at the wavelength of the MALS photometer. Default: 1.3336, calculated for Tris 50 mM, NaCl 100 mM, pH 7.4 (TBS), at 20 °C and at λ=664 nm.

  • Global dn/dc [mL/g]: enter here the differential refractive index increment, in mL/g, for your sample. This value is required to convert raw R(θ) data to I#(t) and I*(q) data, the latter used to calculate a molecular weight from MALS data alone of MALS+SAXS combined data. It is also used to convert refractive index values as provided by a Differential Refractive Index detector, collected during a chromatographic separation, into time-dependent concentration values [mg/mL]. If Gaussian decomposition is used to resolve not-baseline separate peaks, individual dn/dc values can be entered for each decomposed peak during the re-generation of I*(q) MALS data (see ...). Default: 0.189 mL/g (calculated Bovine Serum Albumin (BSA) value in TBS at 20 °C; average value for proteins is 0.185 mL/g).

  • Global extinction coefficient [mL mg^-1 cm^-1]: this value is needed to convert absorbance values as provided by an UV-VIS detector at a specific wavelength (usually 280 nm) during a chromatographic separation, into time-dependent concentration values [mg/mL]. If Gaussian decomposition is used to resolve not-baseline separate peaks, individual extinction coefficients can be entered for each decomposed peak during the re-generation of I*(q) MALS data (see ...). Default: 0.65 [mL mg-1 cm-1] (calculated value for BSA in TBS buffer at 280 nm and for a pathlength of 1 cm).

  • Default concentration [mg/mL]: this is a specific entry for kinetic data, which are usually acquired by mixing reagents attaining a final known concentration that remains constant during the MALS data acquisition. Enter an appropriate value, in mg/mL, for your sample. This field is left blank for HPLC-type data, for which time-dependent concentrations for each MALS frame are usually determined by a separate concentration detector. Default: blank.

  • DLS detector number: some MALS photometers offer the option of a Dynamic Light Scattering (DLS) (also known as Quasi-Elatic Light Scattering (QELS)) detector replacing one of the Static light scattering photodiodes. This entry allows the MALS module to discard on loadng meaningless data that are nowadays exported into csv-style files containg the Rayleigh ratios as a function of time for each detector. Default: 14, the position for the QELS fiber optics in the Wyatt Technology, part of Waters Corporation) now discontinued DAWN HELEOS-II MALS photometer used at the SWING beamline of the Synchrotron SOLEIL, Gif-sur-Yvette, France.

Pressing Quit will exit without retaining any input, pressing Keep will exit keeping any input entered but not saving it in a file, pressing Save will exit keeping any input entered and saving it later on, when exiting the program, in the global US-SOMO configuration file (somo.config).
