SOMO SAS MALS+SAXS Module Options

Last updated: September 2024

This Options currently has only two entries, both under the label Miscellaneous options.

  • The Save CSV transposed checkboxis allows saving the *.csv files produced in the MALS+SAXS module in transposed format. Default: unchecked.

  • the Interpolation method is applied on the I#,I*(q) common times operation (see here). Three methods are available:

    Linear, Quadratic, and Cubic spline (Default: Linear).

Pressing Quit will exit from this module without saving any changes. Pressing Ok will exit saving any changes made.
