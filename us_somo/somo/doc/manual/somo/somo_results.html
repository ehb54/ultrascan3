SOMO - Hydrodynamic Results Module:

Last updated:

In this Panel, a selection of the hydrodynamic parameters computed for the bead model is presented. A full report is saved in a file, which can be viewed by pressing the Load full hydrodynamics results file button.
All the parameters are computed for standard conditions, that is, water @ 20°C. The sedimentation coefficient is computed only if the partial specific volume (vbar) of the molecule is either automatically calculated from the composition or a value in entered in the appropriate field in the Miscellaneous SOMO Options panel. In addition, a file containing the details of the ASA computations can be accessed by pressing the Load ASA Results File button. The View Bead Model File button will instead allow to visualize a previously-generated bead model file.
