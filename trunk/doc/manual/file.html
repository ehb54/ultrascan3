File Menu System:

File Menu:

  1. File:Dataset Info:Velocity Data [Equilibrium Data]

    This function will call up a file dialog that allows you to select an edited dataset of either velocity or equilibrium data. The program will the invoke the Run Details window. For more information, please consult the Run Diagnostics for either velocity or equilibrium data.

  2. File:Export Data:Velocity Data [Equilibrium Data]

    This function allows you to export the edited experimental data to an ASCII spreadsheet suitable for import into another plotting program. Executing this function will also add the exported data and a dataplot to a report, should a Report be generated. For more information on this function, please refer to the Export Data function.

  3. File:Print Data:Velocity Data [Equilibrium Data]

    This function allows you to print the edited experimental data to a printer or a postscript file. For more information on this function, please refer to the Print Data function.

  4. File:Generate Report:Velocity Data [Equilibrium Data] [Equilibrium Fitting Project] [Monte Carlo Project]

    This function allows you to generate an html report of all data analysis results associated with a particular velocity or equilibrium fitting project, or a Monte Carlo project. For more information, please refer to the html Report function.

  5. File:Configuration

    This function allows you to configure various properties of the UltraScan software, such as paths to UltraScan system components, colors, fonts, etc. For more information on this function please consult the Configuration documentation.

  6. File:Administrator

    This function allows you to set the Administrator password. For more details, please refer to the Administrator help file.
