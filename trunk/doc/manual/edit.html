Edit Menu System:

Edit Menu:

  1. Velocity Data:Absorbance Data [Interference Data] [Fluorescence Data] [Edit Cell ID's]

    These functions allow you to edit experimental velocity data as acquired by the Optima XLA/I analytical ultracentrifuge. More information on editing absorbance velocity data is available here, for information on editing velocity data acquired with the interference optical system click here, for information on editing velocity data acquired with the Aviv fluorescence optical system click here. The cell descriptions of edited data can be modified after editing is completed by selecting "Edit Cell ID's". More information on this option can be found here.

  2. Equilibrium Data:Absorbance Data [Interference Data] [Fluorescence Data] [Edit Cell ID's]

    These functions allow you to edit experimental equilibrium data as acquired by the Optima XLA/I analytical ultracentrifuge. More information on editing absorbance equilibrium data is available here, for information on editing equilibrium data acquired with the interference optical system click here, for information on editing equilibrium data acquired with the Aviv fluorescence optical system click here. The cell descriptions of edited data can be modified after editing is completed by selecting "Edit Cell ID's". More information on this option can be found here.

  3. View/Edit Multiwavelength Data

    These functions allow you to view and edit experimental multiwavelength data acquired by the second generation multiwavelength detector developed at the Max Planck Institute for Colloids and Interfaces in Golm, Germany, by Helmut Cölfen's group. More information on editing absorbance equilibrium data is available here.
