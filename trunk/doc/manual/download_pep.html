Peptide Download Form

In order to estimate the partial specific volume, V-bar, of a protein, UltraScan needs the primary amino acid sequence information and the temperature for the experiment. The sequence can be directly imported from the Expasy Swiss Protein Sequence Database. Download and save the sequence as a plain text file with the extension *.pep in your UltraScan root directory (see: Configuration). UltraScan can utilize peptide sequences in the format provided by this database. Then load the saved sequence into the V-bar calculation dialog.
