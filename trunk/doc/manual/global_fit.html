Global Fitting Menu System:

Global Fit:

These functions allow you to access the global fitting modules of UltraScan.

  1. Global Equilibrium Fit:
    Use the "Global Equilibrium Fit" function to simultaneously analyze multiple equilibrium scans from the same experiment. More information about this analysis can be found     here.

  2. Global Extinction Fit:
    With the "Global Extinction Fit" function you can generate extinction profiles from wavelength scans. Multiple wavelength scans are simultaneously fitted to a global model describing the extinction variation with wavelength. More information about this analysis can be found here.

  3. Parallel Cluster Interface:
    This module allows you to control and use a parallel Linux cluster for the analysis of global experiments using the 2-dimensional spectrum analysis and the genetic algorithm analysis. More information can be found here.
