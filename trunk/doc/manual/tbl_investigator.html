Investigator Database Commitment Panel:

This function can be used to store contact information for the investigator to the database (for multiuser facilities the ability to search experimental data by the last name of the investigator is often helpful).

Note: This program will utilize the database backend of UltraScan. This functionality allows you to save certain files and data in a database, among them the nucleotide files analyzed with this module. If the database system is not supported or available on your system, you will get a warning message when opening this module.

To check if an investigator name is already stored in the database, enter the last name in the "Last Name" field and click on "Check Last Name". If the name is not found, an error message will be shown. In that case, enter all contact information and click on "Commit" to store the record in the database. Clicking on "Check Last Name" when no name has been entered in the "Last Name" field causes all available records to be retrieved and displayed in the listbox.

Explanation of fields and buttons:

  • Investigator ID: This is the ID number of the record in the database
  • Reset: Resets all fields to accept new input
  • Update: Allows you to update an existing record
  • Commit: Commit newly entered contact information to the database
  • Delete: Delete the selected nvestigator record from the database. This function requires administrator privileges.
  • Help: Display this help information
  • Exit: Exit the program
