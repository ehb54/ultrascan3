UltraScan Menu System:

All UltraScan functions can be accessed from the popup menus on the main screen. Clicking on any word in the menu bar will open a popup menu with further choices.

Please select an item from the list below for explanations of the purpose and function of each menu item:

  • File Menu
  • Edit Menu
  • Velocity Menu
  • Equilibrium Menu
  • Global Fitting Menu
  • Utilities Menu
  • Simulation Menu
  • Database Menu
  • Help Menu
