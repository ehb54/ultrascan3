Equilibrium Menu:

These functions allow you to access the equilibrium modules of UltraScan.

  1. Suggest Best Speed
    This function allows you to estimate one or more appropriate speeds for an equilibrium experiment. More information about this analysis can be found     here.

  2. Estimate Equilibrium Times
    This function allows you to estimate the length of time required to reach equilibrium. Multiple speeds can be simulated successively and preset or custom rotor positions can be used. More information about this function can be found here.

  3. Global Fit
    Use the "Global Fit" function to simultaneously analyze multiple equilibrium scans from the same experiment. More information about this analysis can be found here.

  4. Ln(C) vs. r2
    The Ln(C) vs. r2 analysis can be used as a first order check on the quality of equilibrium data. The analysis doesn't require any non-linear least squares fitting, and provides some rudimentary statistics, such as standard deviation, correlation coefficient and runs analysis. More information about this analysis can be found here.
