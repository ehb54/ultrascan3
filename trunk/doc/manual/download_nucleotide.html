DNA/RNA Sequence Download Form

UltraScan can estimate the molecular weight of a DNA or RNA sequence. The sequence can be directly imported from the Genbank Entrez Sequence Database. Download and save the sequence as a plain text file with the extension *.seq in your UltraScan root directory (see: Configuration). You can set the NCBI browser to provide you with the correct format by changing the "Display" parameters in the search screen. The correct display parameters should be set as follows:


Then select the desired sequence from the search result and click on "Save" to save a copy to your local disk.

UltraScan can utilize nucleotide sequence files in the format provided by the Genbank database. After downloading and saving as "text", load the saved sequence into the Molecular Weight Calculator.
