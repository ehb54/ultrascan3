Detailed Description

Beckman Raw data scan.

This is the structure of a Beckman raw data file. The file types are:

  • I intensity
  • P interference
  • R absorbance
  • W multi-wavelength
  • F fluorescence

For multi wavelength data, the wavelength and radius data are in opposite positions.
