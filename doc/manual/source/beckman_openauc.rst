==================================
Beckman to OpenAUC Data Converter
==================================


.. toctree:: 
  :maxdepth: 3

.. contents:: Index
  :local: 

The Beckman to OpenAUC converter module converts legacy formatted (.tar.gz) files generated by the Beckman web application for import and conversion into OpenAUC formatted files.

.. image:: /images/beck_converter.png
    :align: center

.. rst-class:: 
    :align: center

    **Beckman (.tar.gz) to OpenAUC Data Converter**

Process:
==========

1. Click on "Load File (tar.gz) in the GUI dialog and navigate to the $HOME/Downloads/RunID*.tar.gz file.

2. Click on "Save"
