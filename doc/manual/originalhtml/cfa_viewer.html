UltraScan CFA Viewer:

AUC data collected with the CFA analytical ultracentrifuge can be viewed and exported to openAUC format for import to UltraScan with this application.

( MANUAL ENTRY UNDER CONSTRUCTION )

Main Window:

Dialog Items:

  • (CFA data directory): Documentation of the local directory in which CFA data resides.
  • Load Raw CFA Data: Open a Load Raw CFA dialog
  • Load US3 AUC Data: Open a Load AUC-from-CFA dialog
  • Data Details: Open a Raw CFA Statistics text dialog or a AUC-from-CFA Statistics text dialog, depending on the type of data loaded.
  • Help Display this documentation.
  • Close Exit the application.
